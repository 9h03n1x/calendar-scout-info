# calendar-scout-info

This repository hosts the info pages for the OAuth Apps for Microsoft and Google.

## Enabling GitHub Pages Deployment

Before the GitHub Actions workflow can deploy to GitHub Pages, you need to enable Pages in the repository settings:

1. Go to your repository on GitHub
2. Click **Settings** > **Pages**
3. Under "Build and deployment", select **GitHub Actions** as the Source
4. Save the settings

Once enabled, the workflow will automatically deploy on every push to the `main` branch.

## Pages

- **index.html** - Main landing page
- **privacy.html** - Privacy policy
- **terms.html** - Terms of service
- **contact.html** - Contact information
